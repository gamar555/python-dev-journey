# 🔁 Day 4 - Python Loops

This folder includes practical examples of `for` loops, `while` loops, and loop control in real-life company workflows.

---

## 🔍 Topics Covered
- `for` and `while` loops
- `break` and `continue`
- `range(start, stop)`
- OTP retry systems
- Invoice number generators
- Looping with filters
